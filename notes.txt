REST API
----------
http://localhost:8000/

GET
    /projectName/             : home page of project
    /projectName/issue/[list] : list of issues
        optional query string :
        can=1&q=searchtext&colspec=Stars+ID+Type+Status+Priority+Owner+Summary
        can=2&q=label%3Atyty%20status%3Astarted%20cc%3Ayyyyyyyyyyyuuuu
        
    /projectName/issue/N      : details of issue N
    /projectName/issue/search : search form
    /projectName/issue/new    : form for adding a new issue
    /projectName/user/John Smith : details of user John Smith
    /projectName/login        : login form
    /projectName/structure    : form for modifying database structure
        add/delete field
        add/delete values in predefined lists

POST
    /projectName/issue/new    : add new issue
    /projectName/issue/N      : modify issue N
    /projectName/user/John Smith : modify user John Smith's details
    /projectName/structure    : modify database structure
